Snapshots Object¶

Snapshots provide consistent read-only views over the entire state of the key-value store.

Simple Usage¶

Snapshots API¶

LevelDB.Snapshot(db)¶

Construct a snapshot based on the current LevelDB state.

Snapshot.Set()¶

Set snapshot.

Snapshot.Reset()¶

Reset snapshot to the current state.

Snapshot.Release()¶

Release snapshot
